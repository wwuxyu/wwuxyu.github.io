eight

GDB

准备工作

  • 关闭编译器的优化选项(-O)
  • 打开调试选项(-g).需要将代码信息打包到可执行程序中。非二进制代码，文本代码。

按回车是默认调用上次执行的命令

启动、退出和帮助等

启动仅指启动gdb程序，并没有去执行待调试的可执行文件。

# 启动和退出
gdb execute_name
quit / q  # 退出

# 给程序传参和显示传参
set args 10 20 
show args

# 帮助命令
help
help list # 具体命令

查看代码

# 查看当前文件代码
list/l        # 默认从main()函数显示好像。
list/l 行号    #
list/l 函数名  #

# 查看非当前文件代码，不可只写文件名(啥文件都不行)
list/l 文件名:行号
list/l 文件名:函数名

# 设置代码显示的行数,默认10行
set listsize/list 行数
show listsize/list

断点操作

# 设置断点
b/break 行号
b/break 函数名
b/break 文件名:行号
b/break 文件名:函数

# 查看断点
i/info b/break

# 删除断点
d/del/delete 断点编号

# 设置断点无效
dis/disable 断点编号

# 设置断点生效
ena/enable 断点编号

# 设置条件断点(通常用在循环中)
b/break 行号 if i = 5

调试命令

# 运行程序
start   # 程序停在第一行
run     # 遇到断点才停

# 继续运行，直到下一个断点停止
continue / c

# 向下执行一行代码(不会进入函数体)
next / n

# 向下单步调试(遇到函数进入函数体)
step / s
finish # 跳出函数体

#变量操作
print / p 变量名  # 打印变量值 
ptype 变量名      # 打印变量类型
set var 变量名=变量值 #给变量赋值
# 自动变量操作
display num      # 自动打印指定变量的值
info/i display   # 显示已经设置的自动变量
undisplay num    # 取消自动变量

# other
until            # 跳出循环
